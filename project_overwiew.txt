Project Description (600 chars)
Developed a real-time industrial safety monitoring application for a steel manufacturing yard that detects workers, trucks, and crane hooks inside hazardous zones. The system fuses five RTSP camera streams with a 3D LiDAR sensor (TCP/UDP) and runs YOLO object detection via ONNX Runtime on GPU. A PySide6 desktop GUI visualizes ROI safety zones, a live top-down map of detected objects, and a Normal/Warning/Danger status with alarm popups. Features include ROI configuration with zoom/pan, cross-camera duplicate filtering, event-based video recording, SQLite-backed settings, and alarm history. (598 chars)
